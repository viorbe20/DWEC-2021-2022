Challenge 4. Es hora de poner el árbol de navidad en casa.

  1. Para ello vamos a crear una función que recibe la altura del árbol, que será un entero positivo del 1 a, como máximo, 100.
  2. Creamos un triángulo de asteriscos * con la altura proporcionada y, a los lados, usamos el guión bajo _ para los espacios. Es muy importante que nuestro árbol siempre tenga la misma longitud por cada lado.
  3. Todos los árboles, por pequeños o grandes que sean, tienen un tronco de dos líneas de #.
  4. Ten en cuenta que el árbol es un string y necesitas los saltos de línea \n para cada línea para que se forme bien el árbol.
