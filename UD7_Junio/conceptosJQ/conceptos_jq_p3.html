$("selector").prop()

  • Booleano. Devuelve true o false respecto a las propiedades del elemento seleccionado.
  • Solo devuelve EL VALOR DEL PRIMER ELEMENTO COINCIDENTE.
  • Con dos parámetros, establece valor.
  • Uso: propiedades DOM (selectedIndex, tagName, nodeName, nodeType, ownerDocument, defaultChecked, defaultSelected )
  • Para atributos HTML se usa attr().
//Get property
$("selector").prop("checked"); //True or false

//Set property
$("selector").prop("disabled", "true");
Ejemplo W3School

Índice

$("selector").removeProp()

  • Elimina la propiedad del elemento seleccionado.
$("selector").removeProp("disabled");
Ejemplo W3School

Índice

$("selector").fadeIn()

  • Hace que el elemento se esconda con un efecto de desvanecimiento.
//No parameters
$("selector").fadeIn();

//Duration
$("selector").fadeIn(milliseconds | "slow" | "fast");

//Easing. 
$("selector").fadeIn("swing" | "linear");
Ejemplo W3School

Índice

$("selector").fadeOut()

  • Hace que el elemento aparezca con un efecto de desvanecimiento.
//No parameters
$("selector").fadeOut();

//Duration
$("selector").fadeOut(milliseconds | "slow" | "fast");

//Easing. 
$("selector").fadeOut("swing" | "linear");
Ejemplo W3School

Índice

$("selector").fadeTo()

  • Permite dar opacidad al desvanecimiento (valor entre 0 y 1).
  • Son obligatorios los parámetro de duración: modo y velocidad.
  • Valores: "slow", "fast", o millisegundos.
$("selector").fadeTo("slow", 0.15);
Ejemplo W3School

Índice

$("selector").fadeToggle()

  • Alterna entre fadeIn() y fadeOut().
  • Permite los parámetro de duración: modo y velocidad.
  • Valores: "slow", "fast", o millisegundos.
//No parameters
$("selector").fadeToggle();

//Parameters
$("selector").fadeToggle(3000);
Ejemplo W3School

Índice

$("selector").css()

  • Permite realizar diferentes operaciones con los attributos css.
//Get value of property
$("selector").css("background-color"); //rgb() value

//Set value
$("selector").css("color", "red");
Ejemplo W3School

Índice

$("selector").css({})

  • Permite establecer varias propiedades al mismo tiempo.
$(selector).css({
"background-color": "green", 
"font-size": "20px"
});
Ejemplo W3School
