4. Métodos de Attributes en jQuery (Conceptos)

Volver a Conceptos jQuery

Índice

  1. $("selector").attr('attribute')
  2. $("selector").attr('attribute', 'value')
  3. $("selector").attr({literal object})
  4. $("selector").removeAttr('attribute')

$("selector").attr("attribute")

  • Devuelve el valor del atributo del elemento seleccionado.
  • Para propiedaes booleanas se usa prop() 
$("selector").attr('height'); //devuelve el valor del alto
Ejemplo W3School

Índice

$("selector").attr("attribute", "value")

  • Establece los valores del atributo seleccionado. 
$("selector").attr('height', "300");
Ejemplo W3School

Índice

$("selector").attr({})

  • Permite establecer al mismo tiempo varios atributos y valores.
$("selector").attr({width: "150", height: "100", color: "green"});
Ejemplo W3School

Índice

$("selector").attr({})

  • Permite eliminar uno o varios atributos y valores.
$("selector").removeAttr("id class");
Ejemplo W3School
