$("parentSelector").append(childElement)

  • Añade contenido al final del elemento seleccionado.
  • En el contenido puede haber: elementos HTML, objetos jQuery y elementos DOM
//HTML elements
$("selector").append("<p>Esto es un párrafo</p>");

//jQuery objects
$("selector").append($("<p></p>").text("Texto con jQuery");

//DOM elements
const txt = document.createElement("p");
txt.innerHTML = "Texto como elemento DOM"; 
$("selector").append(txt);
Ejemplo W3School

$("childSelector").appendTo(parentElement)

  • Añade un elemento hijo a un elemento padre.
  • El contenido debe tener etiquetas HTML.
  • Si el contenido existe, se insertará al final del elemento seleccionado.
$("<span>Hello World!</span>").appendTo("p");

$("selector").each(function)

  • Permite aplicar una función a cada elemento de un iterable.
  • Se puede personalizar en función de los métodos utilizados.
//$("selector").each(function(index,element));
//index - Posición del  selector
//element - Element actual (se puede usar 'this')
                
$("p").each(function(){
    $("div").text($(this).text())
});
Ejemplo W3School

$("selector").length vs. $("selector").size()

  • size() DEPRECATED. En su lugar usar $("selector").length
  • Permite contar los elementos de ese selector.
//HTML
<p>para-1</p>
<p>para-2</p>
<p>para-3</p>

//JS
console.log($("p").length //3
Ejemplo W3School

$("selector").data()

  • Añade y recupera información de un elemento.
//Set information
$("selector").data("animal", "cat");

//Get information
$("selector").data("animal");

//Remove information
$("selector").removeData("animal");
