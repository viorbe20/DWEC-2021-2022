$(":typeName")

  • Los dos puntos selecciona los <input> cuyo tupo haya sido indicado.
  • Posibilita aplicar el mismo evento a varios elementos con ese tipo, de una misma vez.
$(":input") => Selecciona todos los elementos <input>. 
$(":text") => Selecciona todos los <input type="text">. 
$(":radio") => Selecciona todos los <input type="radio">. 
$(":submit") => Selecciona todos los <input type="submit">. 
$(":button") => Selecciona todos los <input type="button">. 
$(":password") => Selecciona todos los <input type="password">.
Ejemplo W3School
  • También posibilita seleccionar <input> por propiedades.
$(":enabled") => Selecciona todos los elementos <input "enabled">.
$(":disabled") => Selecciona todos los <input "disabled">.
$(":selected") => Selecciona todos los <input "selected">.
$(":checked") => Selecciona todos los <input "checked">.
Ejemplo W3School

$(selector).html()

  • Lectura. $(selector).html() devuelve el contenido (text + HTML etiquetas) de los elementos seleccionados.
$(selector).html(); //text + HTML etiquetas
Ejemplo W3School
  • Escritura. Incluimos una cadena a los paréntesis que se añadirá al elemento seleccionado.
$(selector).html("<h1>Nombre</h1>Virginia");
Ejemplo W3School
  • Mediante una función, devuelve el contenido de los elementos seleccionados.
$(selector).html(function(index){
return "<h1>Índice: </h1> " + index;

}); //"Índice: 3"
Ejemplo W3School

$(selector).text()

$(selector).evento1().evento2()

  • Permite añadir varios eventos diferentes al mismo elemento mediante el encadenamiento.
$("selector")
.mouseover(function () { //event 1 
$("selector").html(); 
})
.mouseout(function () { //event 2
$("selector").html(); 
});

$(selector).click

  • Dispara el elemento click en el elemento seleccionado.
$("selector").click(function (e) { 
e.preventDefault(); 
location.href = "index.html"; 
});

Índice

$(selector).addClass

  • Añade UNA O MÁS CLASES a un elemento, permitiendo así dar estilo a diferentes propiedades.
  • No elimina las clases existentes.
  • Si se añaden dos o más, se separan los nombres de las clases con espacios.
$("selector").addClass("className");
Ejemplo W3School

Índice

$(selector).removeClass

  • Elimina las clases pasada por parámetro a un elemento.
  • Si no tiene parámetros, se eliminan TODAS LAS CLASES.
  • Si se eliminan dos o más, se separan los nombres de las clases con espacios.
$("selector").removeClass("className");
$("selector").removeClass(); //remove all
Ejemplo W3School

Índice

$(selector).toggle()

  • Alterna los métodos hide() y show() en el elemento seleccionado.
$("selector").toggle();
Ejemplo W3School
  • El método se puede personalizar pasándole diferentes parámetros.
// $(selector).toggle(speed)
$("selector").toggle(milliseconds | "slow" | "fast");

// $(selector).toggle(easing)
$("selector").toggle("swing"); //slow, fast, slow
$("selector").toggle("linear"); //constant speed

// $(selector).toggle(callback)
$("button").click(function(){
    $("p").toggle(1000, function(){
    console.log("Toggle has finished");
});
Ejemplo W3School

Índice

$(selector).hasClass

  • Pasándole como parámetro el nombre de la clase, comprueba si ese elemento contiene esa clase devolviendo verdadero o falso.
$("selector").hasClass("className");
