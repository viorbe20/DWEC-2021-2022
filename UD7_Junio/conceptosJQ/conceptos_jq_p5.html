$("selector").after()

  • Inserta contenido después de los elementos seleccionados.
  • Valores: elementos HTML, objetos jQuery, elementos DOM 
//HTMl elements
const htmlElement= '<p>Inserta este párrafo</p>';
$("selector").after(htmlElement);

//jQuery elements
const jqElement = $("<i></i>").text("Elemento jQuery");
$("selector").after(jqElement);

//DOM elements
const domElement = document.createElement("p");
domElement.innerHTML = "Elemento HTML"; 
$("selector").after(domElement);

$("selector").clone()

  • Copia todos los elementos seleccionados, INCLUYENDO child nodes, texto y atributos.
  • Si en los paréntesis no se incluyen parámetros, por defecto es FALSE.
  • TRUE: especifica que los manejadores de eventos deben copiarse también.
  • FALSE: especifica que los manejadores de eventos NO deben copiarse.
$("selector").clone(true));
Ejemplo W3School

Índice

$("selector").detach()

  • Elimina los elementos seleccionados, INCLUYENDO child nodes. Mantiene data y eventos.
  • Matiene una copia de los elementos eliminados, lo que permite que se puedan reinsertar más adelante.
  • Para eliminar los eventos y data, usa remove().
  • Para eliminar solo el contenido del elemento seleccionado, usa empty().
$("selector").detach());
Ejemplo W3School

Índice

$("selector").empty()

  • Elimina el contenido y los nodos hijos de los elementos seleccionados.
  • MANTIENE el elemento y sus atributos.
  • Matiene una copia de los elementos eliminados, lo que permite que se puedan reinsertar más adelante.
  • Para eliminar los eventos y data, usa remove().
  • Para eliminar solo el elemento, si data ni eventos, usa data().
$("selector").empty());
Ejemplo W3School

Índice

$("selector").height()

  • Devuelve la altura del PRIMER elemento seleccionado.
  • Establece la altura de TODOS los elementos seleccionados.
  • NO INCLUYE padding, border, o margin.
  • La unidad por defecto son px, aunque se puede especificaf en em, pt, etc.
//Get height
$("selector").height();
$(window).height();
$(document).height();

//Set height
$("selector").height(500);
$("selector").height(20em);
$("selector").height(100pt);
Ejemplo W3School

Índice

$("selector").innerHeight()

  • Devuelve el inner height (altura del elemento + padding) del PRIMER ELEMENTO coincidente.
  • INCLUYE padding.
  • NO INCLUYE border, o margin.
$("selector").innerHeight();
$(window).innerHeight();
$(document).innerHeight();

$("selector").offset()

  • Establece o devuelve las coordenadas del primer elemento seleccionado.
  • Devuelve un objeto con dos propiedades: top y left en píxeles.
  • Establece las coordinadas de todos los elementos que coinciden.
$("selector").offset({top: 200, left: 200});
Ejemplo W3School

Índice

$("selector").offsetParent()

  • Devuelve el primer elemento padre posicionado.
  • Un elemento puede posicionarse con jQuery, o con las propiedades de posición de CSS (relative , absolute, fixed).
  • Establece las coordinadas de todos los elementos que coinciden.
$("selector").offsetParent().css("background-color", "red");;
Ejemplo W3School

Índice

$("selector").outerHeight()

  • Devuelve el outer height (altura del elemento + padding) del PRIMER ELEMENTO coincidente.
