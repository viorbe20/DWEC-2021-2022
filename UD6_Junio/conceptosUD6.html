Conceptos Unidad 6

  1. Ejercicio Colorines

    • Privatización del código. Closure.

      • El uso de closures es una forma de emular métodos privados en JavaScript, los cuales nos permiten restringir el acceso al código.

    • Uso de IIFE.

      • Funciones que se ejecutan tan pronto como se definen.

    • Estilos.

      • Para cambiar los estilos se ha utilizado la propiedad style seguido de aquella característica que hemos querido cambiar. En otros casos, simplemente se ha utilizado classList para añadir o quitar estilos.

      Otra propiedad para cambiar los estilo es className, la cual obtiene y establece el valor del atributo class del elemento especificado. Es una forma además de evitar conflictos con la palabra clave "class" existente en muchos de los lenguages que son usados para manipular el DOM.

    • Comportamiento 'cliquea'.

      • Se ha añadido comportamiento a cada uno de los divs permitiendo así cambiar el estilo haciendo click encima de cada uno de ellos.

    • Uso de this.

      • El objeto contexto de JavaScript en el cual se está ejecutando el código actual.

    • Uso de eventos.

      • En este caso se han utilizado métodos para añadir y eliminar eventos a algunos de elementos del árbol DOM.


  2. Ejercicio Fomulario

    • Recorrido del árbol DOM.

      • De forma que se pueda acceder de un elemento a otro. En este caso hemos accedido a los spans desde el input correspondiente, tomando como referencia su proximidad.

    • Botones.

      • Se han usado botones con diferentes funciones como *reset* o *submit* para gestionar los datos del formulario e incluso crear un objeto.

    • Creación de un objeto.

      • Mediante la definición de una clase se crea un objeto con los atributos elegidos por el usuario. Para ellos se ha usadola nueva sintaxis de Javascript.

    • Uso de regex.

      • Para la validación de uno de los campos. Dicha regex se comprueba en el *setter* de la clase para lanzar una excepción en caso de que la regex no se cumpla.

    • Muestra el estado del objeto.

      • A través de la consola, mediante la llamada a un método.
