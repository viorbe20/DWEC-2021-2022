Local Storage

  • Establece y obtiene almacenamiento local en pares de clave/valor.
  • Almacena datos que no expiran. No se borran cuando se cierra el navegador.
//Precedido o no de window
window.localStorage // Storage {length: 0}
localStorage // Storage {length: 0}

//Guardar datos
localStorage.setItem("student1", "Virginia");
//Leer datos
let studentName = localStorage.getItem("student1");
//Eliminar datos
localStorage.removeItem("student1");
//Eliminar todos
localStorage.clear();

Session Storage

  • Establece y obtiene datos en pares de clave/valor. Siempre en el navegador
  • Los datos se borran cuando se cierra el navegador.
//Precedido o no de window
window.sessionStorage // Storage {length: 0}
sessionStorage // Storage {length: 0}

//Guardar datos
sessionStorage.setItem("student1", "Virginia");
//Leer datos
let studentName = sessionStorage.getItem("student1");
//Eliminar datos
sessionStorage.removeItem("student1");
//Eliminar todos
sessionStorage.clear();
