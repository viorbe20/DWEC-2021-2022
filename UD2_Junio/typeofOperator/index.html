Crea un array con distintos ejemplos de datos.

  1. Además de NaN y null incluye todos los primitivos (menos Symbol())
  2. Incluye varios no primitivos (new Date()...), incluidas declaraciones de funciones
  3. Recórrelo con la sentencia for (no el clásico) y utiliza typeof (para indicar si es o no primitivo) e instanceof (para averiguar el tipo de objeto que es (Array, String...))
  4. En caso de ser primitivo, indica: PRIMITIVO - (tipo de primitivo) - valor
  5. En caso de ser objeto, indica: OBJETO - (true) - de tipo Constructor - valor
  6. Utiliza
    1. objeto.constructor.name para el operador instanceof.
    2. template String para evitar la concatenación de cadenas
    3. error.message en caso de tener que capturar una excepción
